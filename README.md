## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/search_movie.git
```

Go to the project directory

```bash
  cd search_movie
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
